# UmaKuma Branding Prompts

The brand prompt is split into two smaller files so GPT Images does not time out or get confused.

## Use These In Order

1. [BRAND_CORE.md](BRAND_CORE.md)
   - Creates the core mascot identity and launch assets.
   - Use this first.
   - Goal: establish the horse, bear, logo, icon pair, duo study scene, and hero image.

2. [BRAND_EXTENDED.md](BRAND_EXTENDED.md)
   - Creates follow-up assets after the core look is approved.
   - Use this second.
   - Goal: stickers, ride scene, ad image, social banner, and alternate expressions.

## Practical Workflow

- Ask GPT Images for one asset at a time.
- Use PNG-first generation.
- If transparency fails, ask for plain white background and we can remove it later.
- Do not ask for SVG during image generation; vectorize later only if needed.
